First you need to run the supply cabinet.sql file to create the database.

Then, import the project into the workspace of Eclipse

If you are required to link mysql connector to your development environment, use the mysql-connector-java-8.0.27.zip which I have provided in the source code folder

For detail on how to link an external java library or jar file to your environment development, you can google it. 

After that, run the SupplyCabinet.java file to test the program.
